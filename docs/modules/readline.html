Module readline

Readline implementation This library provides a fairly complete readline function.

It supports line editing, history, and obfuscation. Loading this with require will provide only a single function, readline.

Tables

rlopts
Readline options. All fields in this table are optional.

Fields:

  • prompt string A prompt to display
  • history table A table of history
  • noexit boolean Do not exit on ^D
  • exit function A function to call instead of os.exit when ^D is pressed, unless noexit is set
